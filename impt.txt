note: I compiled this using Xcode 7.3.1 on 10.14.6; you need to patch xcode by running ./patch.sh while in the "xcode7mojave" folder

brew unlink binutils
./mach bootstrap
./mach build
if fails: ./mach clobber and do it again
./getcxx.sh
./mach package
open the dmg in obj-i386-apple-darwin10/dist
copy app somewhere
./patchcxx.sh Nightly.app
